# SpaceZero v1.0.0

System Junk Scan and Clean Tool

> **⚠️ Important Notice**
> 
> This is open source code. You need to install Python environment to run it. Use at your own risk.

## Features

- 🗑️ **Smart Scan** - Automatically scan temporary files, log files, cache files
- 🛡️ **Safe Delete** - Move files to Recycle Bin, recoverable anytime
- 📊 **Real-time Progress** - Show scan progress percentage
- ⚡ **Fast & Efficient** - Skip system directories to avoid lag
- 🎯 **Accurate Detection** - Support multiple junk file types

## Usage

1. **Run the Software**
   - Double-click `SpaceZero.exe` to launch
   - Agree to user agreement on first run

2. **Scan Junk Files**
   - Set scan path (default: user directory)
   - Select scan types (Temp Files, Log Files, Cache Files)
   - Click "Start Scan" button

3. **Clean Files**
   - Select files to clean from scan results
   - Click "Clean Selected" button
   - Files will be moved to Recycle Bin

## System Requirements

- OS: Windows 10/11
- No installation required, portable application

# SpaceZero v1.0.0 (中文)

- 🗑️ **智能扫描** - 自动扫描临时文件、日志文件、缓存文件
- 🛡️ **安全删除** - 文件移动到回收站，可随时恢复
